# Methodology

Step 1 does not define production scoring, ingestion, or rendering methodology. It establishes the metadata and safety expectations that later methodology must satisfy.

Future methodology must document:

- source selection criteria
- license and allowed-use review
- data acquisition process
- checksum and provenance tracking
- normalization and deduplication logic
- spatial precision rules
- confidence scoring
- sensitivity handling
- reproducibility controls

Derived scores, including the AI compute score, must be versioned and reproducible from documented inputs.
